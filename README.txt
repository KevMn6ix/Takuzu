Takuzu est un jeu inspiré du sudoku mais est constitué que de 1 et de 0
Aussi appelé Binairo puzzle 

Membres de l'equipe :
    Kevin MUNTO
    Ilyes KHEROUA
    
Comment jouer : 
    - Dans une ligne, il doit y avoir autant de 0 que de 1
    - Dans une colonne, il doit y avoir autant de 0 que de 1
    - Il ne peut pas y avoir deux lignes identiques dans une grille
    - Il ne peut pas y avoir deux colonnes identiques dans une grille
    - Dans une ligne ou une colonne, il ne peut y avoir plus de deux 0 ou deux 1 à la suite 


Pour bien executer le projet takuzu,
Ouvrez les fichiers suivants :
    - main.c

    - Partie1.c
    - Partie1.h

    - Partie2.c
    - Partie2.h

    - Partie3.c
    - Partie3.h

    - fonctions.c
    - fonctions.h

    - TakuzuRules.c
    - TakuzuRules.h

    - CMakeLists.txt
    - cmake-build-debug

Appuyer sur Run

Utilisation :
    - Pour la ligne, saisir un nombre entre 1 et la taille requise
    - Pour la colonne , saisir un nombre entre 1 et la taille requise
